Phase 1

In the first phase of our IT Factory you’ll become acquainted with the IT world and acquire a thorough knowledge of the different IT domains. You’ll learn to develop applications in Python and Java and become an expert in web design. You’ll analyse data with SQL and develop a solid foundation in computer networks and operating systems.

You’ll learn by doing and immediately apply the newly acquired knowledge and skills in various projects. This way, you won’t only acquire technical expertise but also communication skills and soft skills.
